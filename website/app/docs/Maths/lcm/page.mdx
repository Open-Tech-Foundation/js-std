## Syntax

```ts
import { lcm } from '@opentf/std';

lcm(a: number, b: number): number;
```

## Parameters

| Name | Type     | Description          |
| ---- | -------- | -------------------- |
| a    | `number` | The first integer.   |
| b    | `number` | The second integer.  |

## Returns

The least common multiple.

## Examples

```ts
lcm(4, 6) //=> 12
lcm(21, 6) //=> 42
lcm(0, 5) //=> 0
```
