## Syntax

```ts
import { variance } from '@opentf/std';

variance<T>(arr: T[], cb?: (val: T, index: number) => number): number;
```

## Returns

The population variance.

## Examples

```ts
variance([2, 4, 4, 4, 5, 5, 7, 9]) //=> 4
```
