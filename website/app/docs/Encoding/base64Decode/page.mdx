🔠 Decodes a Base64 string.

## Syntax

```ts
import { base64Decode } from '@opentf/std';

base64Decode(str: string): string
```

## Parameters

- `str`: The Base64 string to decode.

## Returns

The decoded string.

## Examples

```ts
base64Decode('SGVsbG8=') //=> 'Hello'
```
