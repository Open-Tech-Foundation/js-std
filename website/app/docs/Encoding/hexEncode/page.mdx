🔢 Encodes a string to Hex format.

## Syntax

```ts
import { hexEncode } from '@opentf/std';

hexEncode(str: string): string
```

## Parameters

- `str`: The string to encode.

## Returns

The Hex encoded string.

## Examples

```ts
hexEncode('Hello') //=> '48656c6c6f'
```
