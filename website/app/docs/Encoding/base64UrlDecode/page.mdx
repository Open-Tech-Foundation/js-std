🔠 Decodes a Base64 URL-safe string.

## Syntax

```ts
import { base64UrlDecode } from '@opentf/std';

base64UrlDecode(str: string): string
```

## Parameters

- `str`: The Base64 URL-encoded string to decode.

## Returns

The decoded string.

## Examples

```ts
base64UrlDecode('aGVsbG8_d29ybGQ=') //=> 'hello?world'

base64UrlDecode('aGVsbG8_d29ybGQ') //=> 'hello?world'
```
