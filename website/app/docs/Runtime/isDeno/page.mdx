Checks if the current runtime is Deno.

## Syntax

```ts
import { isDeno } from '@opentf/std';

isDeno(): boolean
```

## Examples

```ts
isDeno() //=> true (in Deno)

isDeno() //=> false (in Node.js, Bun, browser)
```
