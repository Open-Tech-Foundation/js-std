Checks if the current runtime is Bun.

## Syntax

```ts
import { isBun } from '@opentf/std';

isBun(): boolean
```

## Examples

```ts
isBun() //=> true (in Bun)

isBun() //=> false (in Node.js, Deno, browser)
```
