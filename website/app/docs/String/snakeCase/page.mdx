🐍 Converts string to snake case.

## Syntax

```ts
import { snakeCase } from '@opentf/std';

snakeCase(str: string): string;
```

## Parameters

- `str`: The string to convert.

## Returns

The snake-cased string.

## Examples

```ts
snakeCase('Foo Bar') //=> 'foo_bar'

snakeCase('fooBar') //=> 'foo_bar'

snakeCase('--foo-bar--') //=> 'foo_bar'
```
