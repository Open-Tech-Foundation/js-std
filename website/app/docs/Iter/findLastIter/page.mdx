🧩 Finds the last item that matches a predicate.

## Syntax

```ts
import { findLastIter } from '@opentf/std';

findLastIter<T>(iter: Iterable<T>, fn: (val: T) => boolean): T | undefined
```

## Parameters

- `iter`: The source iterable.
- `fn`: The predicate function.

## Returns

The last matching item, or undefined.

## Examples

```ts
findLastIter([1, 2, 3, 2], (x) => x === 2) //=> 2
```
