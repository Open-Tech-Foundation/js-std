🧩 Yields items as long as a predicate is true.

## Syntax

```ts
import { takeWhileIter } from '@opentf/std';

takeWhileIter<T>(iter: Iterable<T>, fn: (val: T) => boolean): IterableIterator<T>
```

## Parameters

- `iter`: The source iterable.
- `fn`: The predicate function.

## Returns

A new iterable iterator.

## Examples

```ts
[...takeWhileIter([1, 2, 3, 4], (x) => x < 3)] //=> [1, 2]
```
