🆔 Generates a time-sortable UUID v7 using capability-detected crypto.

<Callout type="info">
  Uses Web Crypto when available and falls back to \`node:crypto.webcrypto\` in Node-compatible environments.
</Callout>

## Syntax

```ts
import { uuidv7 } from '@opentf/std';

uuidv7(): string;
```

## Returns

A random UUID v7 string.

## Examples

```ts
uuidv7() //=> e.g. '0195e2a4-8c3d-7000-8000-123456789abc' (time-ordered)
```
