🔢 Generates an array of cryptographically secure random bytes.

Uses Web Crypto when available and falls back to `node:crypto.webcrypto`
in Node-compatible environments.

## Syntax

```ts
import { randomBytes } from '@opentf/std';

randomBytes(size: number): Uint8Array;
```

## Parameters

- `size`: The number of bytes to generate.

`size` must be a non-negative integer.

## Returns

A `Uint8Array` containing the random bytes.

## Examples

```ts
randomBytes(16) //=> Uint8Array(16) [ ... ]
```
