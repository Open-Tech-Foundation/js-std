🔢 Generates a cryptographically secure random integer between min (inclusive) and max (inclusive).

Uses Web Crypto when available and falls back to `node:crypto.webcrypto`
in Node-compatible environments.

## Syntax

```ts
import { randomInt } from '@opentf/std';

randomInt(min: number, max: number): number;
```

## Parameters

- `min`: The minimum value (inclusive).
- `max`: The maximum value (inclusive).

Both bounds must be integers.

## Returns

A random integer.

## Examples

```ts
randomInt(0, 100) //=> 42
randomInt(1, 10) //=> 7
```
