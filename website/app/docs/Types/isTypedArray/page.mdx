Checks if the given value is [TypedArray](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/TypedArray).

## Syntax

```ts
import { isTypedArray } from '@opentf/std';

isTypedArray(value: unknown): val is TypedArray
```

## Examples

```ts
isTypedArray([]) //=> false

isTypedArray(new Int8Array()) //=> true

isTypedArray(new Float64Array()) //=> true
```
