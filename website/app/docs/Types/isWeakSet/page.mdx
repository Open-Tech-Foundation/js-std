Checks if the given value is [WeakSet](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/WeakSet).

## Syntax

```ts
import { isWeakSet } from '@opentf/std';

isWeakSet(value: unknown): boolean;
```

## Examples

```ts
isWeakSet(new Set()) //=> false

isWeakSet(new WeakSet()) //=> true
```
