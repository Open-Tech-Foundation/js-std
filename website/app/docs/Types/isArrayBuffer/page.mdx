Checks if the given value is an [ArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer).

## Syntax

```ts
import { isArrayBuffer } from '@opentf/std';

isArrayBuffer(val: unknown): val is ArrayBuffer
```

## Examples

```ts
isArrayBuffer([]) //=> false

isArrayBuffer(new ArrayBuffer(8)) //=> true
```
