🔢 Checks if a value is a finite number.

<Callout type="info">
  It only returns `true` for actual `number` values, excluding `NaN` and `Infinity`.
</Callout>

## Syntax

```ts
import { isNumber } from '@opentf/std';

isNumber(value: unknown): val is number
```

## Parameters

- `value`: The value to check.

## Returns

`true` if the value is a finite number, `false` otherwise.

## Examples

```ts
isNumber(1) //=> true
isNumber(1.5) //=> true
isNumber(NaN) //=> false
isNumber(Infinity) //=> false

isNumber('1') //=> false
isNumber('1_000') //=> false

// Other types
isNumber(null) //=> false
isNumber(true) //=> false
isNumber([]) //=> false
```
