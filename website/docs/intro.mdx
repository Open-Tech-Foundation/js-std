# @open-tech-world/js-utils

> A Collection of JavaScript Utility Functions.

## Requirements

- It supports Evergreen browsers only.

- Node.js >=14.0.0

## Installation

<Tabs
  defaultValue="npm"
  values={[
    {label: 'npm', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>
  <TabItem value="npm">

```shell
npm install @open-tech-world/js-utils
```

</TabItem>
  <TabItem value="yarn">

```shell
yarn add @open-tech-world/js-utils
```

  </TabItem>
</Tabs>

## Usage

```ts
import { utilityFn } from '@open-tech-world/js-utils';

// ⚠️ Here, the utilityFn is a placeholder for the actual utility function name.
// For example: arrayDiff, cloneObj, camelCase, etc.
utilityFn();
```

:::tip
The collections are grouped into Array, Maths, Object, String, Timers, and Types, respectively.
:::
