# camelCase

Converts string to camelCase.

## Syntax

```ts
import { camelCase } from '@opentf/std';

camelCase
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| str | `string` | The string to convert. |

## Returns

`string`: The camelCased string.

## Example

```js
camelCase('Foo Bar') //=> 'fooBar'
```
