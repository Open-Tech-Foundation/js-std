# pascalCase

Converts string to PascalCase.

## Syntax

```ts
import { pascalCase } from '@opentf/std';

pascalCase(str: string): string
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| str | `string` | The string to convert. |

## Returns

`string`: The PascalCased string.

## Example

```js
pascalCase('foo bar') //=> 'FooBar'
```
