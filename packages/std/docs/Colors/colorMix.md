# colorMix

Mixes two colors together.

## Syntax

```ts
import { colorMix } from '@opentf/std';

colorMix( color1: any, color2: any, weight = 0.5, format: any = 'hex', ): any
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| color1 | `any` | - The first color. |
| color2 | `any` | - The second color. |

## Returns

`any`: - The mixed color.
