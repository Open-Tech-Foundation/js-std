# colorIsLight

Checks if a color is light.

## Syntax

```ts
import { colorIsLight } from '@opentf/std';

colorIsLight(input: any): boolean
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |

## Returns

`boolean`: - True if the color is light.
