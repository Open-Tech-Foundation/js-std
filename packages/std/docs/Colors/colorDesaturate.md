# colorDesaturate

Decreases the saturation of a color.

## Syntax

```ts
import { colorDesaturate } from '@opentf/std';

colorDesaturate( input: any, amount: number, format: ColorFormat = 'hex', ): any
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |
| amount | `number` | - The amount to desaturate (0 to 1). |

## Returns

`any`: - The desaturated color.
