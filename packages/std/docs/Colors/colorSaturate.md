# colorSaturate

Increases the saturation of a color.

## Syntax

```ts
import { colorSaturate } from '@opentf/std';

colorSaturate( input: any, amount: number, format: ColorFormat = 'hex', ): any
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |
| amount | `number` | - The amount to saturate (0 to 1). |

## Returns

`any`: - The saturated color.
