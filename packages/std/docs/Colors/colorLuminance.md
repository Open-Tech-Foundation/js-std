# colorLuminance

Calculates the relative luminance of a color.

## Syntax

```ts
import { colorLuminance } from '@opentf/std';

colorLuminance(input: any): number
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |

## Returns

`number`: - The relative luminance (0 to 1).
