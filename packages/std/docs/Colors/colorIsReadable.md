# colorIsReadable

Checks if the contrast between two colors meets WCAG standards.

## Syntax

```ts
import { colorIsReadable } from '@opentf/std';

colorIsReadable( color1: any, color2: any, level: AccessibilityLevel = 'AA', ): boolean
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| color1 | `any` | - The first color. |
| color2 | `any` | - The second color. |
| level | `AccessibilityLevel` | - The WCAG level to check against (default: 'AA'). |

## Returns

`boolean`: - Whether the colors are readable.
