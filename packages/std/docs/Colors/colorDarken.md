# colorDarken

Decreases the lightness of a color.

## Syntax

```ts
import { colorDarken } from '@opentf/std';

colorDarken( input: any, amount: number, format: ColorFormat = 'hex', ): any
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |
| amount | `number` | - The amount to darken (0 to 1). |

## Returns

`any`: - The darkened color.
