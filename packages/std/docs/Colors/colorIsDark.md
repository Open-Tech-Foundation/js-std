# colorIsDark

Checks if a color is dark.

## Syntax

```ts
import { colorIsDark } from '@opentf/std';

colorIsDark(input: any): boolean
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| input | `any` | - The input color. |

## Returns

`boolean`: - True if the color is dark.
