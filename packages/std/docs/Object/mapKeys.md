# mapKeys

Creates an object with the same values and keys generated by running each
own enumerable string keyed property of object through the callback.

## Syntax

```ts
import { mapKeys } from '@opentf/std';

mapKeys<T extends object>( obj: T, fn: (value: T[keyof T], key: string) => string, ): Record<string, T[keyof T]>
```

## Example

```js
mapKeys({ a: 1, b: 2 }, (v, k) => k + v) //=> { a1: 1, b2: 2 }
```
