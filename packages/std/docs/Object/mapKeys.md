# mapKeys

Creates an object with the same values and keys generated by running each
own enumerable string keyed and symbol keyed property of object through the
callback.

When the source object uses `Object.create(null)`, the returned object preserves
that null prototype.

### Example

```js
mapKeys({ a: 1, b: 2 }, (v, k) => k + v) //=> { a1: 1, b2: 2 }
```
