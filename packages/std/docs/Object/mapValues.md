# mapValues

Creates an object with the same keys and values generated by running each
own enumerable string keyed property of object through the callback.

## Syntax

```ts
import { mapValues } from '@opentf/std';

mapValues<T extends object, R>( obj: T, fn: (value: T[keyof T], key: string) => R, ): Record<keyof T, R>
```

## Example

```js
mapValues({ a: 1, b: 2 }, (v) => v * 2) //=> { a: 2, b: 4 }
```
