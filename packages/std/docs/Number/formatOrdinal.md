# formatOrdinal

Formats a number to its ordinal form (1st, 2nd, 3rd, etc.).

@param {number} n The number to format.
@returns {string} The ordinal formatted string.

### Example

```js
formatOrdinal(1) //=> '1st'
formatOrdinal(2) //=> '2nd'
formatOrdinal(3) //=> '3rd'
formatOrdinal(11) //=> '11th'
formatOrdinal(21) //=> '21st'
```
