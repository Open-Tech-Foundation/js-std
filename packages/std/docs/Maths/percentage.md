# percentage

It returns percentage of value & total.

## Syntax

```ts
import { percentage } from '@opentf/std';

percentage(value: number, total: number): number
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| value | `number` | - The current value |
| total | `number` | - The total value |
