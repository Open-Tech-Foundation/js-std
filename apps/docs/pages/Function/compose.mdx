> 🧪 Performs function composition from right to left.

<Callout type="default">
  The `compose` function is equivalent to `a(b(c(val)))`.
</Callout>

## Syntax

```ts copy
import { compose } from '@opentf/std';

compose(val: unknown, ...fns: ((arg: any) => any)[]): unknown
```

## Parameters

- `val`: The initial value to pass into the composition.
- `...fns`: A sequence of functions to apply from right to left.

## Returns

The final result of the composition.

## Examples

```ts
compose(1); //=> 1

compose(-1, Math.abs); //=> 1

compose(-4, Math.sqrt, Math.abs); //=> 2

compose(
  1.5,
  Math.ceil,
  (x) => x + 1,
  (x) => x * 5
); //=> 9 ( (1.5 * 5) + 1 = 8.5 -> ceil = 9 )
```

## Learn

Why do we need function composition?

- **Readability**: Deep nesting of functions like `c(b(a(x)))` is hard to read.
- **Conciseness**: It eliminates the need for temporary variables.
- **Flexibility**: Method chaining is limited when dealing with `await`, `yield`, etc.
