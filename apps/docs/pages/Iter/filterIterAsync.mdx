# filterIterAsync

Filters items based on an async/sync predicate.

## Usage

```js
import { filterIterAsync } from '@opentf/std';

async function* asyncGen(arr) {
  for (const item of arr) {
    yield item;
  }
}

const result = await filterIterAsync(asyncGen([1, 2, 3]), x => x % 2 === 0);
// ...
```
