> 🧩 Maps each item and flattens the result.

## Syntax

```ts copy
import { flatMapIter } from '@opentf/std';

flatMapIter<T, U>(iter: Iterable<T>, fn: (val: T) => Iterable<U>): IterableIterator<U>
```

## Parameters

- `iter`: The source iterable.
- `fn`: The mapper function returning iterables.

## Returns

A new iterable iterator.

## Examples

```ts
[...flatMapIter([1, 2], (x) => [x, x * 10])] //=> [1, 10, 2, 20]
```
