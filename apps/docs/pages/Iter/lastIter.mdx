> 🧩 Returns the last item from an iterable.

## Syntax

```ts copy
import { lastIter } from '@opentf/std';

lastIter<T>(iter: Iterable<T>): T | undefined
```

## Parameters

- `iter`: The source iterable.

## Returns

The last item, or undefined.

## Examples

```ts
lastIter([1, 2, 3]) //=> 3
```
