# firstIterAsync

Returns the first item from an async iterable.

## Usage

```js
import { firstIterAsync } from '@opentf/std';

async function* asyncGen(arr) {
  for (const item of arr) {
    yield item;
  }
}

const result = await firstIterAsync(asyncGen([1, 2, 3]));
// ...
```
