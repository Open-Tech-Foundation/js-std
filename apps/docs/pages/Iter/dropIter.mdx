> Returns a Generator that skips the first n items from an Iterable.

<Callout type="info">
Allows you to "jump" into a stream after a certain number of elements.
</Callout>

## Syntax

```ts
import { dropIter } from "@opentf/std";

dropIter(iterable: Iterable<any>, n: number)
```

## Examples

```ts
function* gen() {
  yield 1;
  yield 2;
  yield 3;
  yield 4;
}

const remaining = dropIter(gen(), 2);
Array.from(remaining); //=> [3, 4]
```
