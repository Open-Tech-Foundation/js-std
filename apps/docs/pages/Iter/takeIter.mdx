> Returns a Generator that yields the first n items from an Iterable.

<Callout type="info">
Works on any object that follows the iterator protocol (Arrays, Sets, Maps, and custom Generators).
</Callout>

## Syntax

```ts
import { takeIter } from "@opentf/std";

takeIter(iterable: Iterable<any>, n: number)
```

## Examples

```ts
function* gen() {
  yield 1;
  yield 2;
  yield 3;
}

const limited = takeIter(gen(), 2);
Array.from(limited); //=> [1, 2]
```
