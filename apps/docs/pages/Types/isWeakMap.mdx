> Checks if the given value is [WeakMap](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/WeakMap).

## Syntax

```ts
import { isWeakMap } from '@opentf/std';

isWeakMap(value: unknown): boolean;
```

## Examples

```ts
isWeakMap(new Map()) //=> false

isWeakMap(new WeakMap()) //=> true
```
