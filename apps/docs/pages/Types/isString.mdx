> Checks if the given value is [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String).

## Syntax

```ts
import { isString } from '@opentf/std';

isString(value: unknown): boolean;
```

## Examples

```ts
isString('abc') //=> true

isString(String(1)) //=> true

isString(1) //=> false
```
