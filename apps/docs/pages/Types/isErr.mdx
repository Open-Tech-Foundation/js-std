> Checks if the given value is an [Error](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error).

## Syntax

```ts
import { isErr } from '@opentf/std';

isErr(val: unknown): boolean;
```

## Examples

```ts
isErr(new Error()) //=> true

isErr({}) //=> false
```
