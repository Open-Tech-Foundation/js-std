> Converts an RGBA object to a hex string.

## Syntax

```ts
import { colorToHex } from '@opentf/std';

colorToHex(color: { r: number, g: number, b: number, a: number }): string;
```

## Examples

```ts
colorToHex({ r: 255, g: 0, b: 0, a: 1 }) //=> '#ff0000'

colorToHex({ r: 255, g: 0, b: 0, a: 0.5 }) //=> '#ff000080'
```
