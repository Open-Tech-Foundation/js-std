> Parses a color string into an RGBA object.

## Syntax

```ts
import { colorParse } from '@opentf/std';

colorParse(str: string): { r: number, g: number, b: number, a: number } | null;
```

## Examples

```ts
colorParse('#ff0000') //=> { r: 255, g: 0, b: 0, a: 1 }

colorParse('rgb(255, 0, 0)') //=> { r: 255, g: 0, b: 0, a: 1 }

colorParse('rgba(255, 0, 0, 0.5)') //=> { r: 255, g: 0, b: 0, a: 0.5 }

colorParse('hsl(0, 100%, 50%)') //=> { r: 255, g: 0, b: 0, a: 1 }
```
