> Converts an RGBA object to an hsl/hsla string.

## Syntax

```ts
import { colorToHsl } from '@opentf/std';

colorToHsl(color: { r: number, g: number, b: number, a: number }): string;
```

## Examples

```ts
colorToHsl({ r: 255, g: 0, b: 0, a: 1 }) //=> 'hsl(0, 100%, 50%)'

colorToHsl({ r: 255, g: 0, b: 0, a: 0.5 }) //=> 'hsla(0, 100%, 50%, 0.5)'
```
