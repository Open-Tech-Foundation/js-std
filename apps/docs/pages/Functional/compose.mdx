# compose

Performs functions composition from right to left.

<Callout type="info">
  This is a polymorphic utility that supports both **Synchronous** and **Asynchronous** functions.
  If any function in the chain returns a Promise, the remaining chain is automatically awaited.
</Callout>

## Usage

```ts
import { compose } from "@opentf/std";

// Synchronous
const transform = compose(
  (n) => n * 2,
  (x, y) => x + y
);
transform(1, 2); //=> 6

// Asynchronous
const asyncTransform = compose(
  (n) => n * 2,
  async (x, y) => x + y
);
await asyncTransform(1, 2); //=> 6
```

## Signature

```ts
function compose(
  ...fns: ((...args: any[]) => any)[]
): (...args: any[]) => any;
```

### Parameters

- `fns`: A series of functions to compose. The right-most (first executed) function can accept multiple arguments; subsequent functions must be unary.

## Examples

### Execution order

```ts
const result = compose(
  (x) => x + 1, // (10) + 1 = 11
  (x) => x * 2  // (5) * 2 = 10
)(5); //=> 11
```
