> 😶 The "No Operation" function. It does nothing and returns `undefined`.

## Syntax

```ts copy
import { noop } from '@opentf/std';

noop(): void
```

## Parameters

None.

## Returns

`undefined`.

## Examples

```ts
noop(); //=> undefined

function test(cb) {
  // Use noop as a default callback to avoid "cb is not a function" errors
  const fn = cb || noop;
  fn();
}

test(); // Safely calls the noop function
```
