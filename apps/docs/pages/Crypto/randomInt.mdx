> 🔢 Generates a cryptographically strong random integer within a range.

## Syntax

```ts copy
import { randomInt } from '@opentf/std';

randomInt(min: number, max: number): number
```

## Examples

```ts
randomInt(1, 10) //=> 7
randomInt(0, 100) //=> 42
```
