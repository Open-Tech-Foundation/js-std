> 🆔 Generates a random alphanumeric ID.

## Syntax

```ts copy
import { randomId } from '@opentf/std';

randomId(size: number = 21): string;
```

## Parameters

- `size`: The length of the ID. Default: `21`.

## Returns

A random alphanumeric string.

## Examples

```ts
randomId() //=> 'Kj7L2n9P1m3Q5r7S9t1U3'
randomId(10) //=> 'A1B2C3D4E5'
```
