> 🐢 Controls the minimum execution time for a function.

<Callout type="info">
  Useful for preventing UI "flickering" when an async operation is too fast.
</Callout>

## Syntax

```ts copy
import { paceRun } from '@opentf/std';

paceRun<T>(
  func: () => Promise<T>,
  ms: number
): Promise<T>;
```

## Parameters

- `func`: The asynchronous function to execute.
- `ms`: The minimum duration in milliseconds.

## Returns

A Promise that resolves with the function result, but only after at least `ms` have elapsed.

## Examples

```ts
// Even if fetchData takes 10ms, paceRun ensures it takes at least 500ms
const data = await paceRun(fetchData, 500);
```
