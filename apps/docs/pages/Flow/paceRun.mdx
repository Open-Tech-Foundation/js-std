# paceRun

Creates a throttled version of a function that only invokes the original function at most once per every `interval` milliseconds.

<Callout type="info">
  Useful for limiting the rate of execution for high-frequency events like `scroll` or `mousemove`.
</Callout>

## Usage

```ts
import { paceRun } from "@opentf/std";

const run = paceRun((val) => {
  console.log("Processing:", val);
}, 1000);

run(1); // Executes immediately
run(2); // Queued
run(3); // Replaces 2, will execute after 1000ms
```

## Signature

```ts
function paceRun<T extends (...args: any[]) => any>(
  func: T,
  interval?: number,
  options?: {
    leading?: boolean;
    trailing?: boolean;
  }
): {
  (...args: Parameters<T>): void;
  cancel: () => void;
  flush: () => void;
  pending: () => boolean;
};
```

### Parameters

- `func`: The function to throttle.
- `interval`: The number of milliseconds to throttle invocations to. Default: `0`.
- `options`:
    - `leading`: Execute on the leading edge of the interval. Default: `true`.
    - `trailing`: Execute on the trailing edge of the interval. Default: `true`.

### Methods on returned function

- `cancel()`: Cancels any pending execution.
- `flush()`: Immediately invokes the pending execution and cancels the timer.
- `pending()`: Returns `true` if there is a pending execution.
