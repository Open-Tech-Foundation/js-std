> 📦 Batches multiple calls to an asynchronous function into a single execution.

<Callout type="info">
  This is useful for bundling frequent updates (like database writes or API calls) into batches.
</Callout>

## Syntax

```ts copy
import { batchRun } from '@opentf/std';

batchRun<T, R>(
  func: (batch: T[]) => Promise<R[]>,
  options?: {
    maxBatchSize?: number;
    delay?: number;
  }
): (item: T) => Promise<R>;
```

## Parameters

- `func`: The batch processor function. It receives an array of items and should return an array of results.
- `options`:
    - `maxBatchSize`: The maximum number of items in a single batch.
    - `delay`: The time to wait for more items before processing a batch (ms).

## Returns

A function that takes an item and returns a Promise for its specific result.

## Examples

```ts
const saveUser = batchRun(async (users) => {
  return await db.bulkSave(users);
}, { maxBatchSize: 10, delay: 50 });

// These will be batched together
saveUser({ name: 'Alice' });
saveUser({ name: 'Bob' });
```

## Try

<REPL code={`const { batchRun } = require('@opentf/std');

async function demo() {
  const processor = batchRun(async (batch) => {
    console.log('Processing batch of size:', batch.length);
    return batch.map(n => n * 2);
  }, { delay: 100 });

  const results = await Promise.all([
    processor(1),
    processor(2),
    processor(3)
  ]);
  console.log('Results:', results);
}

demo();
`} />
