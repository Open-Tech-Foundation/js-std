> 🔁 Retries an asynchronous function based on a specified retry policy.

## Syntax

```ts copy
import { retryRun } from '@opentf/std';

retryRun<T>(
  func: () => Promise<T>,
  options?: {
    retries?: number;
    delay?: number;
    onRetry?: (error: unknown, attempt: number) => void;
  }
): Promise<T>;
```

## Parameters

- `func`: The asynchronous function to retry.
- `options`:
    - `retries`: The maximum number of retry attempts. Default: `3`.
    - `delay`: The delay between retries in milliseconds. Default: `1000`.
    - `onRetry`: A callback invoked on each retry attempt.

## Returns

A Promise that resolves with the function result if successful.

## Examples

```ts
const data = await retryRun(fetchData, {
  retries: 5,
  delay: 2000,
  onRetry: (error, attempt) => {
    console.log(\`Retry attempt \${attempt} failed: \${error.message}\`);
  }
});
```

## Try

<REPL code={`const { retryRun } = require('@opentf/std');

async function demo() {
  let count = 0;
  const result = await retryRun(async () => {
    count++;
    if (count < 3) throw new Error('Fail');
    return 'Success';
  }, { retries: 5, delay: 100 });
  console.log(result);
}

demo();
`} />
