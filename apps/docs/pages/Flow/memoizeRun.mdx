> 🧠 Memoizes a function, caching its results based on input arguments.

<Callout type="info">
  Supports custom cache keys and Time-To-Live (TTL) expiration.
</Callout>

## Syntax

```ts copy
import { memoizeRun } from '@opentf/std';

memoizeRun<T extends (...args: any[]) => any>(
  func: T,
  options?: {
    resolver?: (...args: Parameters<T>) => any;
    ttl?: number;
  }
): T;
```

## Parameters

- `func`: The function to memoize.
- `options`:
    - `resolver`: A function to generate a unique cache key from the arguments.
    - `ttl`: Time-to-live in milliseconds. After this time, the cached result is invalidated.

## Returns

A memoized version of the input function.

## Examples

```ts
const heavyTask = (n) => {
  console.log('Computing...');
  return n * 2;
};

const memoized = memoizeRun(heavyTask, { ttl: 1000 });

memoized(5); // Computing... 10
memoized(5); // 10 (from cache)
```

## Try

<REPL code={`const { memoizeRun, sleep } = require('@opentf/std');

async function demo() {
  const task = (n) => {
    console.log('Calculating...');
    return n + 10;
  };
  const memoized = memoizeRun(task, { ttl: 200 });

  console.log(memoized(1)); // Calculates
  console.log(memoized(1)); // Cached
  await sleep(300);
  console.log(memoized(1)); // Calculates again (TTL expired)
}

demo();
`} />
