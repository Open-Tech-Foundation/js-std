> 💤 Executes a function during the browser's idle periods.

<Callout type="info">
  In Node.js, this falls back to \`setImmediate\`.
</Callout>

## Syntax

```ts copy
import { idleRun } from '@opentf/std';

idleRun<T>(
  func: () => T | Promise<T>,
  options?: {
    timeout?: number;
  }
): Promise<T>;
```

## Parameters

- `func`: The function to execute when idle.
- `options`:
    - `timeout`: If specified, the function is executed even if the browser hasn't become idle after this many milliseconds.

## Returns

A Promise that resolves with the function result.

## Examples

```ts
await idleRun(() => {
  // Perform non-essential background task
  sendAnalytics();
});
```
