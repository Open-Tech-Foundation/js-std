> 🚦 Limits the number of times a function can be called within a time window.

<Callout type="info">
  Commonly used for API rate limiting or preventing rapid user actions.
</Callout>

## Syntax

```ts copy
import { rateLimitRun } from '@opentf/std';

rateLimitRun<T extends (...args: any[]) => any>(
  func: T,
  options?: {
    limit?: number;
    window?: number;
  }
): T;
```

## Parameters

- `func`: The function to rate limit.
- `options`:
    - `limit`: The maximum number of calls allowed in the window. Default: `1`.
    - `window`: The time window in milliseconds. Default: `1000`.

## Returns

A rate-limited version of the input function.

## Examples

```ts
const limitedLog = rateLimitRun(console.log, { limit: 2, window: 1000 });

limitedLog(1);
limitedLog(2);
limitedLog(3); // This will wait until the next window starts
```
