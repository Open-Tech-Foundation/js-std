> 🗼 Converts a string to [PascalCase](https://en.wikipedia.org/wiki/Camel_case#Upper_camel_case).

## Syntax

```ts copy
import { pascalCase } from '@opentf/std';

pascalCase(str: string): string
```

## Parameters

- `str`: The string to convert.

## Returns

The PascalCased string.

## Examples

```ts
pascalCase('na cl') //=> 'NaCl'

pascalCase('Foo-Bar') //=> 'FooBar'

pascalCase('react_component') //=> 'ReactComponent'

pascalCase('--foo.bar') //=> 'FooBar'

pascalCase('cinema Scope') //=> 'CinemaScope'

pascalCase('the quick brown fox') //=> 'TheQuickBrownFox'
```
