> 🧼 Removes ANSI escape codes from a string.

## Syntax

```ts copy
import { stripANSI } from '@opentf/std';

stripANSI(str: string): string
```

## Examples

```ts
stripANSI('\u001b[31mHello\u001b[0m')
//=> 'Hello'

stripANSI('\x1b[4mUnderline\x1b[0m')
//=> 'Underline'
```
