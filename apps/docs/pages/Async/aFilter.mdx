> ⏳ Filters an array using an asynchronous callback function.

## Syntax

```ts copy
import { aFilter } from '@opentf/std';

aFilter<T>(
  arr: T[], 
  cb: (value: T, index: number) => Promise<boolean>
): Promise<T[]>
```

## Parameters

- `arr`: The source array.
- `cb`: An async function that returns a `Promise<boolean>`.

## Returns

A `Promise` that resolves to a new filtered array.

## Examples

```ts
const arr = [1, 2, 3, 4, 5];

function isEven(n) {
  return new Promise((resolve) => {
    setTimeout(() => resolve(n % 2 === 0), 10);
  });
}

const filteredArr = await aFilter(arr, async (n) => await isEven(n));
console.log(filteredArr); //=> [2, 4]
```

## Why?

- **Native `filter`**: Fails with async callbacks because an async function always returns a `Promise`, which is truthy. As a result, native `filter` will keep every element in the array regardless of the resolved boolean value.

```js
// Native approach (Fails):
const filtered = arr.filter(async (x) => await isEven(x)); 
// Returns all items because every callback returns a Promise object.

// @opentf/std approach (Works):
const filtered = await aFilter(arr, isEven);
```

- **Ease of Use**: `aFilter` correctly awaits the results of each callback before deciding whether to include the element in the final array.
