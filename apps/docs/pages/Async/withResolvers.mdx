> 🔑 Creates a Promise along with its `resolve` and `reject` functions.

<Callout type="info">
  **Legacy Support**: While the native `Promise.withResolvers()` is only available in modern environments (e.g., Node.js 22+), this utility brings that same power to legacy environments like **Node.js 18+** without requiring global polyfills.
</Callout>

## Syntax

```ts copy
import { withResolvers } from '@opentf/std';

const { promise, resolve, reject } = withResolvers<T>();
```

## Parameters

None.

## Returns

An object containing:
- `promise`: A new `Promise` object.
- `resolve`: A function to resolve the promise.
- `reject`: A function to reject the promise.

## Examples

```ts
const { promise, resolve, reject } = withResolvers();

promise.then((val) => console.log(val));

Math.random() > 0.5 ? resolve("Success") : reject("Error");
```
