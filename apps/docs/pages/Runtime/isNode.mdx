> Checks if the current runtime is Node.js.

## Syntax

```ts
import { isNode } from '@opentf/std';

isNode(): boolean
```

## Examples

```ts
isNode() //=> true (in Node.js)

isNode() //=> false (in browser, Bun, Deno)
```
