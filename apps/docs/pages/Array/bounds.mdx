> 📏 Returns a tuple containing the minimum and maximum values of an array.

<Callout type="info">
  If duplicate values are present in the array, the first match will be returned.
</Callout>

## Syntax

```ts copy
import { bounds } from '@opentf/std';

bounds<T>(
  arr: T[] = [],
  by: (val: T) => number = (x: T) => x as unknown as number
): [T, T] | null
```

## Parameters

- `arr`: The source array.
- `by`: An optional function to extract the numeric value for comparison. Defaults to identity function.

## Returns

A tuple `[min, max]` or `null` if the array is empty (after compacting).

## Examples

```ts
bounds([1, 2, 3, 4, 5]) //=> [1, 5]

const arr = [
  {
    name: 'x',
    age: 10,
  },
  {
    name: 'y',
    age: 16,
  },
  {
    name: 'z',
    age: 13,
  },
  { name: 'y2', age: 16 },
];
bounds(arr, (o) => o.age)
//=> 
// [
//  {
//    name: 'x',
//    age: 10,
//  },
//  {
//    name: 'y',
//    age: 16,
//  },
// ]
```
