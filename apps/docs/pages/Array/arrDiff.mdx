> It creates an array with the values, not included in the other array.

<Callout type="warning">
It does not work with non primitive values.
</Callout>

<Callout type="info">
It uses [Object.is()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/is) for equality comparisons.
</Callout>

## Syntax

```ts copy
import { arrDiff } from '@opentf/utils';

arrDiff(arr1: unknown[], arr2: unknown[]): unknown[];
```

## Examples

```ts
arrDiff(); // It throws an error;

arrDiff([1], []); // [1]

arrDiff([1], [1, 2]); // []

arrDiff([1, "a"], [1, 2]); // ['a']

arrDiff([0], [0]); // []

arrDiff([-0], [0]); // [-0]

arrDiff([], []); // [[]]

arrDiff(["apple", "mango", "orange"], ["mango", "apple"]); // ['orange']
```
