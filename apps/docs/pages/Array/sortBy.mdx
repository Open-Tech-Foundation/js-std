> Sorts a list of objects and returns a new array.

<Callout emoji="âœ…" type="info">
Immutable: This does not mutate the given array.
</Callout>

**Related:**

- [sort](/Array/sort)

## Syntax

```ts
sortBy<T>(
  arr: T[],
  ...tuples: [key: string | ((obj: T) => unknown), order: "asc" | "desc"][]
): T[]
```
<Callout type="info">
The `key` in the tuple array must one of object first level props.
</Callout>

## Usage

```ts
import { sortBy } from "@opentf/utils";

sortBy([{...}], ['key', 'asc'], [o => o.prop, 'desc']);
```

## Examples

```ts
const fruits = [
  { name: 'banana', amount: 2 },
  { name: 'apple', amount: 4 },
  { name: 'pineapple', amount: 2 },
  { name: 'mango', amount: 1 },
  { name: 'orange', amount: 5 },
];

sortBy(fruits, ['name', 'asc']);
sortBy(fruits, [(o) => o.name, 'asc']);
//=> [
//   { name: 'apple', amount: 4 },
//   { name: 'banana', amount: 2 },
//   { name: 'mango', amount: 1 },
//   { name: 'orange', amount: 5 },
//   { name: 'pineapple', amount: 2 },
// ]

sortBy(fruits, ['name', 'asc'], ['amount', 'desc']);
sortBy(fruits, ['name', 'asc'], [(o) => o.amount, 'desc']);
//=> [
//   { name: 'orange', amount: 5 },
//   { name: 'apple', amount: 4 },
//   { name: 'banana', amount: 2 },
//   { name: 'pineapple', amount: 2 },
//   { name: 'mango', amount: 1 },
// ]
```
