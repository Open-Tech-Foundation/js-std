> Sorts a list of items and returns a new array.

<Callout emoji="âœ…" type="info">
Immutable: This does not mutate the given array.
</Callout>

<Callout type="info">
The following types are supported:

- Number
- String
- Boolean
- Date
</Callout>

<Callout type="warning">
It cannot sort if the values in the array is of mixed types.
</Callout>

**Related:**

- [sortBy](/Array/sortBy)

## Syntax

```ts
sort<T>(arr: T[], order = 'asc'): T[]
```
<Callout type="info">
The order param can be either `'asc'` or `'desc'`.
</Callout>

## Usage

```ts
import { sort } from "@opentf/std";

sort([...]);
```

## Examples

```ts
sort([1, 3, 2]) //=> [1, 2, 3]

sort(['x', 'z', 'y'], 'desc') //=> ['z', 'y', 'x']
```
