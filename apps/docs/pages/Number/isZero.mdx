> Checks if the given number is positive zero (0).

<Callout type="info">
In JavaScript 0 === -0 is true.
</Callout>

## Syntax

```ts
import { isZero } from '@opentf/utils';

isZero(n: number): boolean;
```

## Examples

```ts
isZero(0) //=> true

isZero(0.0) //=> true

isZero(0x0) //=> true

isZero(-0) //=> false

isZero(-0.0) //=> false

isZero(-0x0) //=> false
```
