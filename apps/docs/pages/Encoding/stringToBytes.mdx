> 🔠 Converts a string to a Uint8Array of bytes.

## Syntax

```ts copy
import { stringToBytes } from '@opentf/std';

stringToBytes(str: string): Uint8Array
```

## Parameters

- `str`: The string to convert.

## Returns

The byte array.

## Examples

```ts
stringToBytes('Hello') //=> Uint8Array [72, 101, 108, 108, 111]
```
