> 🔠 Encodes a string to Base64 URL-safe format.

## Syntax

```ts copy
import { base64UrlEncode } from '@opentf/std';

base64UrlEncode(str: string, options?: { pad?: boolean }): string
```

## Parameters

- `str`: The string to encode.
- `options.pad`: Whether to add padding (default true).

## Returns

The Base64 URL-encoded string.

## Examples

```ts
base64UrlEncode('hello?world') //=> 'aGVsbG8_d29ybGQ='

base64UrlEncode('hello?world', { pad: false }) //=> 'aGVsbG8_d29ybGQ'
```
