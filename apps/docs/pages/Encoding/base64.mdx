> 🔠 Base64 encoding and decoding.

## Syntax

```ts copy
import { base64 } from '@opentf/std';

base64: {
  encode: (input: string | Uint8Array) => string;
  decode: (input: string) => string;
}
```

## Examples

```ts
base64.encode('hello') //=> 'aGVsbG8='
base64.decode('aGVsbG8=') //=> 'hello'
```
