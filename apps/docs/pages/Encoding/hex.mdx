> Hex encoding and decoding.

## Syntax

```ts
import { hexEncode, hexDecode } from "@opentf/std";

hexEncode(str: string)
hexDecode(str: string)
```

## Examples

```ts
const encoded = hexEncode("Hello"); //=> "48656c6c6f"
const decoded = hexDecode(encoded); //=> "Hello"
```
