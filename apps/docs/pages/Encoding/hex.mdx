> 🔢 Hexadecimal encoding and decoding.

## Syntax

```ts copy
import { hex } from '@opentf/std';

hex: {
  encode: (input: string | Uint8Array) => string;
  decode: (input: string) => string;
}
```

## Examples

```ts
hex.encode('hello') //=> '68656c6c6f'
hex.decode('68656c6c6f') //=> 'hello'
```
