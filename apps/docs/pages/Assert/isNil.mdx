> Checks if the given value is Nil / Noting.

<Callout type="info">
The `undefined` & `null` values are considered [Nullish](https://developer.mozilla.org/en-US/docs/Glossary/Nullish).
</Callout>

## Syntax

```ts
import { isNil } from '@opentf/std';

isNil(val: unknown): Boolean;
```

## Examples

```ts
isNil('') //=> false

isNil([]) //=> false

isNil({}) //=> false

isNil() //=> true

isNil(undefined) //=> true

isNil(null) //=> true
```

## Try

<REPL code={`const { isNil } = require('@opentf/std');

const arr = [1, null, 2, undefined, 3]
arr.filter((i) => !isNil(i));
`} />
