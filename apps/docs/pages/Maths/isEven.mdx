> 🔢 Checks if a number is even.

## Syntax

```ts copy
import { isEven } from '@opentf/std';

isEven(n: number): boolean
```

## Parameters

- `n`: The number to check.

## Returns

`true` if the number is even, `false` otherwise.

## Examples

```ts
isEven(0) //=> true

isEven(1) //=> false

isEven(2) //=> true

isEven(-1) //=> false

isEven(-2) //=> true
```
