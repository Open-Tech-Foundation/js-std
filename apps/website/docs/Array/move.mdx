> It moves an array element from one index position to another.

## Syntax

```ts
move(arr: T[], from: number, to: number): T[];
```
## Usage

```ts
import { move } from '@opentf/utils';

move([], 0, 1);
```

## Examples

```ts
move([1, 2, 3], 0, 2) //=> [2, 3, 1]

move([1, 2, 3], 0, 5) //=> [2, 3, 1]

move([1, 2, 3], 5, 0) //=> [1, 2, 3]
```
